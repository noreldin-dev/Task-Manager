# 📝 Smart Task Manager (CLI)
A robust Command Line Interface (CLI) application built with **Python** to help users manage their daily productivity. This project focuses on data persistence and clean user interaction.

## ✨ Features
* **Data Persistence:** Automatically saves and loads tasks using a `tasks_data.json` file.
* **Status Tracking:** Mark tasks as complete or pending with visual indicators (✔/❌).
* **Input Validation:** Robust error handling for invalid inputs and empty tasks.
* **Organized Workflow:** Simple and intuitive menu-driven interface.

## 🛠️ Technical Skills Demonstrated
* **File I/O:** Reading and writing JSON files for permanent storage.
* **List Manipulation:** Efficiently managing and filtering Python dictionaries within lists.
* **Exception Handling:** Using `try-except` blocks to prevent program crashes.
